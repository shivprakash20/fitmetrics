# FitMetrics Web

Next.js web app and backend-for-frontend API for FitMetrics.

## Run

From repository root:

```bash
npm run dev:web
```

Or directly from this folder:

```bash
npm run dev
```

## API namespaces

- `/api/health`
- `/api/v1/profile`
- `/api/v1/analytics`
- `/api/v1/auth/mobile/*` (mobile login, refresh, logout)
